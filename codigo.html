Introducción a la Programación Web

La programación web es el proceso de crear sitios y aplicaciones que funcionan en Internet. Se compone de lenguajes y tecnologías que permiten definir tanto la estructura de una página como su estilo y funcionalidad.

Los dos lenguajes básicos que todo desarrollador web debe conocer son HTML y CSS.

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que define la estructura de un sitio web. Con él se crean títulos, párrafos, enlaces, imágenes, listas y tablas, organizando todo el contenido de la página.

Ejemplo de un código HTML simple:

<h1>Hola Mundo</h1>
<p>Este es un párrafo en HTML.</p>
<a href="https://www.google.com">Ir a Google</a>

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que se utiliza para dar estilo a la estructura creada con HTML. Permite definir colores, fuentes, tamaños, márgenes, bordes y diseños de página.

Algunas propiedades básicas de CSS son:

  • color: cambia el color del texto. Ejemplo: color: blue;
  • background-color: define el color de fondo. Ejemplo: background-color: yellow;
  • font-size: ajusta el tamaño de la fuente. Ejemplo: font-size: 20px;
  • border: agrega bordes alrededor de un elemento. Ejemplo: border: 2px solid black;
  • margin: establece el espacio exterior de un elemento. Ejemplo: margin: 10px;
  • padding: define el espacio interior entre el contenido y el borde. Ejemplo: padding: 15px;

Conclusión

HTML y CSS son la base de cualquier sitio web. Mientras HTML estructura la información, CSS se encarga de darle diseño y estilo. Dominar ambos lenguajes es el primer paso para convertirse en un desarrollador web.
